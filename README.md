# Modeling Navigation in a Headless CMS (Strapi/React) - Part 1

This repo is related to a blog post found at [https://waynelincoln.dev/blog/modeling-navigation-in-a-headless-cms-strapi-react-part-1](https://waynelincoln.dev/blog/modeling-navigation-in-a-headless-cms-strapi-react-part-1)

`git clone https://github.com/wayne-lincoln/strapi-react-headless-navigation.git`

## To run the backend (Strapi)

1. `cd strapi`
2. `npm install`
3. `npm run develop`
4. Create an admin account: [http://localhost:1337/admin](http://localhost:1337/admin)

## To run the frontend (React)

1. `cd frontend`
2. `npm install`
3. `npm start`
4. Server will be runnning at [http://localhost:3000](http://localhost:3000)
